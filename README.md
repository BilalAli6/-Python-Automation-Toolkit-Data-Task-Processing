## ğŸ“ How to Use

1. **Organize files**

   * Enter the folder path you want to organize.
   * Files are moved to subfolders based on file extensions.

2. **Process CSV/JSON**

   * Enter the CSV file path.
   * Optionally enter an output file (CSV/JSON) to save results.
   * Summary statistics will be displayed in the terminal.

3. **Web Scraping**

   * Enter the website URL.
   * Enter a CSS selector (default is `'p'` for paragraphs).
   * Top 10 elements are displayed in the terminal.

4. **Exit**

   * Exit the program safely from the menu.
